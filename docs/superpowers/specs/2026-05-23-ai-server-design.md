# 智算中心服务器全栈知识体系 — 设计文档

## 背景

受众：技术小白，知道 CPU/GPU/内存等概念但不系统。
目标：场景驱动、自顶向下，完整了解智算中心服务器从硬件到软件的全栈知识。
产出：学习路线卡片 + 结构化知识文档（HTML 格式）。

## 学习模型

场景驱动。从"训练一个千亿参数大模型需要什么"出发，每一步反推为什么要用这种硬件/软件。
每阶段回答三个问题：这是什么 → 为什么需要它 → 市场上谁在做。

## 路线结构（6 阶段，12 篇文档）

### 阶段 1：全景图
- 智算中心是什么，和传统数据中心有什么区别
- 训练 vs 推理的工作负载特点

### 阶段 2.1：GPU 芯片
- GPU 架构基础（Tensor Core、TFLOPS 含义）
- 主流芯片：NVIDIA (H100/H200/B200)、AMD (MI300X)、华为 (昇腾 910B)、Intel Gaudi 3
- NVIDIA 的护城河：CUDA 生态

### 阶段 2.2：内存系统
- GPU 显存 HBM：为什么大模型参数必须装进显存，HBM2e→HBM3→HBM3e 演进
- 品牌：SK 海力士、三星、美光
- 服务器主内存 DRAM：DDR5、CXL 内存池化
- 核心概念：模型参数 × 2 字节 ≈ 最低显存需求（FP16）、Memory Wall

### 阶段 2.3：GPU 服务器整机
- NVIDIA DGX / HGX / MGX 体系
- 8-GPU 节点内部拓扑：NVLink + NVSwitch
- OEM 整机厂商：Dell、HPE、联想、浪潮、超微、华为
- 品牌对比：各厂商主力 GPU 服务器型号

### 阶段 3：网络
- 为什么几千张 GPU 之间通信比计算还贵
- InfiniBand vs RoCE（RDMA 原理）
- 网络拓扑：Leaf-Spine、DragonFly
- 芯片与交换机：NVIDIA/Mellanox、Broadcom、华为、Cisco
- 光模块：中际旭创、Finisar、Coherent
- 关键指标：带宽、延迟、All-Reduce 通信

### 阶段 4：存储
- 训练数据量级与吞吐需求
- 并行文件系统：Lustre、GPFS、WekaFS
- 存储分层：NVMe SSD 热数据 → HDD 冷数据
- 厂商：Dell、HPE、华为、Pure Storage、NetApp、DDN

### 阶段 5：散热与供电
- GPU 功耗现实：单卡 700W+，整机 10kW+
- 风冷极限与液冷方案（冷板式、浸没式）
- PUE 与数据中心能效
- 液冷厂商：CoolIT、台达、曙光数创、Vertiv

### 阶段 6.1：驱动与加速层
- CUDA、cuDNN、NCCL

### 阶段 6.2：容器与资源调度
- Docker/Containerd、Kubernetes、Slurm
- 智算中心的作业调度特点

### 阶段 6.3：分布式训练框架
- PyTorch、DeepSpeed、Megatron-LM
- 并行策略：数据并行、张量并行、流水线并行、3D 并行

### 阶段 6.4：推理引擎
- TensorRT-LLM、vLLM、Triton Inference Server

### 阶段 6.5：集群管理与监控
- Prometheus + Grafana、NVIDIA DCGM、BMC/Redfish

### 总结篇
- 整合所有阶段，给出完整参考架构
- 一个真实千卡集群长什么样（架构图 + 配置清单）

## 每篇文档规范

1. 场景导入 — 先讲"为什么需要它"
2. 核心概念 — 白话解释，专业术语首次出现时加注解
3. 技术要点 — 原理层面，控制深度，不讲数学推导
4. 主流产品与品牌 — 市场上主要玩家、产品线、对比
5. 一张图总结 — 收尾回顾
6. 延伸阅读 — 推荐资源链接

每篇 5-8 页阅读量，插图配合说明。首次出现的专业名词给白话解释。

## 产出格式

HTML 文档。包含：
- 左侧导航栏（6 阶段目录）
- 正文区：排版清晰，表格与插图配合
- 品牌对比表：每个硬件阶段附主要厂商对比
- 响应式布局，适合阅读
